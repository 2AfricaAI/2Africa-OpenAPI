# Code of Conduct

## Our commitment

The 2Africa OpenAPI community is open to anyone who wants to help build
a fair, vendor-neutral standard for African agricultural data
exchange — regardless of country, language, employer, technical
background, gender, race, religion, age, or any other personal
attribute.

We pledge to make participation in our project a respectful,
harassment-free experience.

## Our standards

Behaviour that contributes to a positive environment:

- Using welcoming and inclusive language
- Respecting differing viewpoints and experiences
- Accepting constructive criticism gracefully
- Focusing on what is best for the standard and its implementers
- Showing empathy toward other community members

Behaviour that will not be tolerated:

- Personal attacks, insults, or derogatory comments
- Public or private harassment
- Publishing others' private information without explicit permission
- Sustained disruption of discussion or refusal to compromise
- Sexualised language or imagery, or unwelcome sexual attention
- Trolling or deliberate intimidation
- Other conduct which could reasonably be considered inappropriate in
  a professional setting

## Scope

This Code of Conduct applies within all project spaces — the GitHub
repository (issues, pull requests, discussions, code review), the
project mailing list (when established), Steering Committee meetings,
and any public event held in the name of the project.

It also applies when an individual is officially representing the
project in public spaces.

## Enforcement

If you experience or witness unacceptable behaviour, please report it
by emailing **conduct@2africa.ai**. All reports will be:

- Read by the project maintainers and treated as confidential.
- Investigated promptly and fairly.
- Responded to within 7 days, with a description of next steps.

The reporting party will not be named publicly without their consent.

### Consequences

Maintainers and the Steering Committee may, in response to a verified
incident, take any action they deem appropriate, including:

1. A private warning
2. A public warning
3. Temporary suspension from the project's interaction channels
4. Permanent ban from the project

Maintainers who fail to enforce this Code in good faith may face
temporary or permanent removal from the maintainer team, decided by
the Steering Committee.
